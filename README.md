# ReactNative-fitApp

React-Native - Redux fitApp

Demo: [ReactNative-fitApp](https://expo.io/@extwiii/fitapp)

Features
--------
- React Native 
- Create React Native App
- Airbnb Eslint
- Redux
- Redux-thunk
- React-native-scrollable-tab-view
- Expo

Prerequisites
-------------
- make sure you have npm or yarn on your computer
- make sure you have expo app for [iOS](https://itunes.apple.com/app/apple-store/id982107779) or [Android](https://play.google.com/store/apps/details?id=host.exp.exponent) in order to run on your mobile

Getting Started
-------------

The easiest way to get started is to clone the repository:

```bash
# Get the latest snapshot
git clone https://github.com/extwiii/ReactNative-fitApp.git

# Change directory
cd ReactNative-fitApp

# Install dependencies
yarn install

# Then start your app 
yarn start

```
- Scan QR code with your Expo app
